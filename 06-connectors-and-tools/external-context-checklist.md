# External Context Checklist

Use this before asking Codex to read from a connector or MCP server.

- what external source contains the truth?
- is the source authoritative for this question?
- what exact question should the tool answer?
- what local files will the answer affect?
- what data should not be pulled into the session?
- what verification proves the final repo change?

If the answer can be found by reading the repo, stay local.
