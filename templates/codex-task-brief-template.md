# Codex Task Brief

## Goal

What should be true when this task is done?

## Surface

CLI, IDE, web/cloud, GitHub, automation, MCP, connector, or SDK.

## Context

- repository, branch, issue, PR, or source artifact
- relevant files or components
- external sources that must be checked

## Scope

- allowed changes
- explicitly out of scope
- ownership area for delegated or background work

## Verification

- commands to run
- screenshots, logs, review comments, or other evidence to collect

## Expected Output

- diff, explanation, PR summary, comment response, report, or generated artifact
